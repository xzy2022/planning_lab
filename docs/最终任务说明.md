题目：针对阿克曼转向或者差动转向结构的无人平台开展运动规划问题呀牛，已知起始姿态A，给定目标姿态B，计算满足起始姿态、目标姿态、障碍物避碰（考虑平台实际尺寸）、运动学模型等约束的全局路径。

1. 环境部分已知，感知距离有限（例如障碍物环境大小为200m╳200m，感知距离为30m），目标点在感知范围之外，在此条件下，对从起始姿态到目标姿态的路径规划；

2. 无人平台沿规划路径行驶过程中，利用实时更新的感知结果对规划结果调整，保证安全性和目标可达性；

3. 利用基于优化的算法对初始路径进行平滑；

实验要求：

1. 改变环境中障碍物的分布、密度等；

2. 改变起始姿态和目标姿态；

3. 对规划结果进行统计分析（路径平滑性、规划实时性，成功率等）；

4. 编程语言不限制，Python，C++…均可；

5. 形成实验报告（包含要素：方法原理，算法流程，不同算法实验结果对比，实验结论等）。

6. 分别实现RRT和混合A星算法
